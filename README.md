## 🚀 Proje Hakkında

**LunOS**, macOS estetiğinden ilham alan, tamamen web teknolojileri (HTML, CSS, JavaScript) kullanılarak geliştirilmiş modern bir masaüstü simülasyonudur. Şık tasarımı, akıcı animasyonları ve kullanıcı dostu arayüzü ile tarayıcı üzerinde gerçek bir işletim sistemi deneyimi sunar.

---

## ✨ Temel Özellikler

- 🖥️ **Şık Masaüstü:** Dinamik duvar kağıtları ve düzenli ikon yapısı.
- 🔐 **Kilit Ekranı:** Güvenli giriş ve kullanıcı profil yönetimi.
- 🚀 **Launchpad:** Tüm uygulamalara tek bir yerden hızlı erişim.
- 🛠️ **Kontrol Merkezi:** Wi-Fi, parlaklık ve ses ayarlarını anında yönetin.
- 🧩 **Widget Paneli:** Hava durumu, takvim ve notlar gibi özelleştirilebilir araçlar.
- 🔍 **Spotlight Arama:** Dosyalarınızı ve uygulamalarınızı saniyeler içinde bulun.
- 🔔 **Bildirim Sistemi:** Sistem olaylarından anında haberdar olun.

## 🛠️ Kurulum ve Çalıştırma

Projeyi yerel makinenizde çalıştırmak oldukça basittir:

1. Depoyu bilgisayarınıza indirin:
   ```bash
   git clone https://github.com/kullanici/LunOs.git
   ```
2. Proje dizinine gidin:
   ```bash
   cd LunOs
   ```
3. `index.html` dosyasını tarayıcınızda açın veya Python sunucusunu başlatın:
   ```bash
   python start_lunos.py
   ```

## ❓ Sıkça Sorulan Sorular (SSS)

<details>
  <summary><strong>1. LunOS hangi tarayıcılarda çalışır?</strong></summary>
  LunOS, modern tarayıcıların (Chrome, Firefox, Safari, Edge) en güncel sürümlerinde en iyi performansı verecek şekilde optimize edilmiştir.
</details>

<details>
  <summary><strong>2. Kendi uygulamalarımı nasıl ekleyebilirim?</strong></summary>
  `js/apps.js` dosyası üzerinden yeni uygulama tanımlamaları yapabilir ve `uygulamalar/` dizini altına kendi HTML dosyalarınızı ekleyebilirsiniz.
</details>

<details>
  <summary><strong>3. LunOS gerçek bir işletim sistemi mi?</strong></summary>
  Hayır, LunOS bir web tabanlı simülasyondur. Tarayıcı üzerinde çalışan bir arayüz projesidir.
</details>

<details>
  <summary><strong>4. Tasarımı nasıl özelleştirebilirim?</strong></summary>
  `css/system.css` ve `css/window.css` dosyalarındaki CSS değişkenlerini değiştirerek renkleri, köşe yuvarlaklıklarını ve temayı kolayca güncelleyebilirsiniz.
</details>
